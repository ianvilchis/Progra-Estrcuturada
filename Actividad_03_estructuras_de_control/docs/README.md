## Descripción
En esta actividad analizamos y desarrollamos soluciones utilizando diferentes estructuras de control de flujo en lenguaje C. Trabajamos con la evaluación lógica de expresiones mediante operadores de cortocircuito, la implementación de bifurcaciones condicionales para la toma de decisiones y el anidamiento de estructuras para resolver flujos lógicos complejos.

## Temas Aplicados
* **Estructuras de Control Condicionales:** Uso de `if` y `if-else` para desviar el flujo del programa según condiciones específicas (como determinar la mayoría de edad o si un número es par/impar mediante el residuo `%`).
* **Condicionales Anidados:** Implementación de bloques condicionales dentro de otros para evaluar múltiples criterios en la toma de decisiones (lógica para la asignación de becas y estatus escolar).
* **Evaluación de Expresiones:** Control del flujo mediante operadores lógicos (`&&`, `||`), operadores relacionales y el comportamiento de pre/post incrementos en variables.
