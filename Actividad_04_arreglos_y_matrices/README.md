## Descripción
Esta actividad abarca el uso y la manipulación de estructuras de datos lineales y bidimensionales en lenguaje C. Se desarrollaron prácticas para el manejo de cadenas de texto utilizando funciones estándar y, como proyecto principal, se implementó el juego interactivo **Batalla Naval (BattleShip)** empleando matrices para representar los tableros de juego y disparos de dos jugadores.

## Temas Aplicados
* **Arreglos Unidimensionales y Cadenas (`strings`):** Uso de la librería `<string.h>` y funciones como `strlen()` para medir longitud, `strcmp()` para comparar cadenas (validación de claves), `strcat()` para concatenación y `strcpy()` para asignar textos a arreglos de caracteres.
* **Matrices (Arreglos Bidimensionales):** Modelado de tableros de juego (`Tablero[N][M]`) de tipo entero y carácter para posicionar elementos espaciales y registrar estados dinámicos (barcos ocultos, impactos y agua).
* **Modularidad y Modularización:** Estructuración del código mediante funciones especializadas con paso de parámetros por valor y por referencia (punteros para el conteo de impactos).
